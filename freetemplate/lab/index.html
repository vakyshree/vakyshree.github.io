New to my Arsenal

Particle.Js

Particle.js is a light weight javascript framework it lets you create particles effect.It uses GPU processing speed. Also it has a wide range of customizable properties to fullfill your needs in optimum way.

Web Gl

Web Gl is javascript API for rendering 3D graphics within any compatible web browser without the use of plug-ins.WebGL is integrated completely into all the web standards of the browser allowing GPU accelerated usage.

Bootstrap

Bootstrap is a free and open-source front-end web framework for designing websites and web applications. It contains HTML- and CSS-based design templates, as well as optional JavaScript extensions.

Particle.js Experiment

I wanted to check the threshold of particle.js where it starts to lag, quality loss and performance lag. So in this experiment i pushed particles in an increment manner and playing with vibrant colors at the same instant.As my GPU is Nvidia 940MX it took near 2000-3000 particles at the same instant to make it laggy. Although 2000-3000 particles are too much to handle better GPU can perform better than this.I'm well impressed by particle.js parsers.

Solar system Web GL

I have created a solar system using WebGL. It does not support perfect physics and distances in between planets but its a good start to sharpen my Web GL skills.Im using Three.js framework to write Webgl parser and all in all it makes my life easier. Textures and mapping images are taken from open source sites and its fully reusable content. Zooming In and Out is controlled by mouse so you can get into the deatils of the solar system.

Particle vs Frequency

Another particle.js experiment, In this i have taken music albums and played around it.What basically i have done is using music frequency and scaling them with particle numbers (high frequency more particle low frequence less particle likesie).So it gives wave like effect addind colors and distribution makes it beautiful to look at it.Regards to Mrsucidesheep i have used his albums to play around this experiment
